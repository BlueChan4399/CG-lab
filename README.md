# CG-Lab1 - 粒子群交互系统

## 1. 项目简介
本项目是一个基于 **Taichi** 编程语言实现的 2D 粒子系统。通过 GPU 加速，实现了大量粒子（如 1000 个）的实时物理演算与渲染，并支持用户通过鼠标进行实时引力交互。

## 环境要求

- Python >= 3.12
- [uv](https://github.com/astral-sh/uv) 包管理器

```bash
# 安装依赖
uv sync
```
