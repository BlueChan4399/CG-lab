# CG-Lab2-Bézier Curve Rasterization

## 1. 项目简介

本项目基于 **Taichi** 图形框架和 **NumPy** 实现了 Bézier 曲线的交互式绘制与 GPU 光栅化。在基础实验之上，进一步实现了超采样反走样、控制点拖拽、Bézier Spline 插值以及 De Casteljau 包络线动态可视化。核心内容包括：

- **De Casteljau 算法**：手动实现 Bézier 曲线的递归线性插值采样
- **GPU 逐像素光栅化**：将连续曲线离散化为屏幕像素，在内核中并行写入显存
- **Batching 优化**：在 CPU 端预计算全部采样点坐标，一次性批量发往 GPU，避免频繁的 PCIe 小数据通信
- **对象池技巧**：利用固定容量的 Taichi Field 动态渲染数量随鼠标点击变化的控制点与控制多边形
- **超采样反走样**：通过 2× 超采样 + 均值下采样平滑锯齿
- **Bézier Spline**：基于 Catmull-Rom 思想，将插值点自动转换为 C1 连续的分段三次 Bézier 曲线
- **包络线动画**：动态展示 De Casteljau 算法逐层插值的构造过程

## 2. 效果展示

### Demo：交互绘制 Bézier 曲线

通过鼠标左键点击画布添加控制点，程序会实时用灰色线段绘制控制多边形，并在 GPU 端用白色像素点渲染对应的 Bézier 曲线。当控制点数量不少于 2 个时，窗口会实时显示绿色包络线动画。按下键盘 `空格` 可开启/关闭反走样，鼠标 `右键` 可在 Bézier 模式与 Spline 模式间切换。

## 3. 项目架构

```
Work2/
├── __init__.py        # 模块初始化文件
├── config.py          # 配置参数（窗口、采样数、对象池容量、颜色）
├── bezier.py          # De Casteljau 算法、包络线层级、Spline 转换
├── main.py            # 主程序入口：事件循环、GPU 内核、光栅化与渲染
└── README.md          # 项目说明文档
```

| 文件 | 职责 |
|------|------|
| `config.py` | 定义窗口分辨率 `1024×1024`、最大控制点数、采样点数、反走样倍数、颜色常量 |
| `bezier.py` | 实现 `de_casteljau` 单点采样、`sample_bezier` 批量采样、`de_casteljau_levels` 包络层级、`catmull_rom_to_bezier` Spline 转换 |
| `main.py` | 初始化 Taichi GPU、声明显存池、处理鼠标/键盘事件、执行光栅化内核、调用 GUI 实时渲染 |

## 4. 实现功能

### 基础功能
- **De Casteljau 算法**：在 Python 端通过循环插值递归计算 Bézier 曲线上的点
- **批量光栅化**：预分配 `1000×2` 的采样坐标数组，通过 `draw_batch_kernel` 让 GPU 并行点亮像素
- **单点像素内核**：同时保留 `draw_pixels_kernel(pixels, x, y)` 以符合实验任务要求
- **边界安全**：在内核中对 `px`、`py` 进行越界检查，防止访问非法显存地址
- **对象池绘制**：使用固定长度 `100` 的 Taichi Vector Field，将未使用的控制点填充为 `-10.0` 隐藏到屏幕外，实现动态数量的控制点渲染

### 选做功能
- **反走样**：按 `空格` 键开启 `AA:ON`，通过 2× 超采样光栅化后均值下采样，有效减少曲线的锯齿边缘
- **控制点拖拽**：按住鼠标左键靠近已有控制点（半径 15 像素内）即可拖动；松开左键自动固定位置
- **Bézier Spline**：按鼠标右键切换到 `Mode=Spline`，此时控制点被视为插值点，程序自动生成分段三次 Bézier 曲线，保证连接处 C1 连续
- **包络线动画**：当控制点不少于 2 个时，窗口实时用绿色线段绘制 De Casteljau 算法的逐层插值过程，并以红色小圆点标示当前动画在曲线上的位置

## 5. 运行方式

### 运行程序

在项目根目录（`CG-lab`）下执行：

```bash
uv run -m src.Work2.main
```

### 交互操作

| 按键/鼠标 | 功能 |
|-----------|------|
| `鼠标左键` | 空白处点击添加控制点；在已有控制点附近按住拖拽移动 |
| `鼠标右键` | 在 Bézier 模式与 Spline 模式间切换 |
| `空格` | 开启/关闭反走样（AA） |
| `C` | 清空所有控制点，恢复空白画布 |
| `ESC` | 退出程序 |

### 注意事项

- 程序默认使用 `arch=ti.gpu` 启动 Taichi；若显卡不支持 CUDA/Vulkan，请修改 `main.py` 中的 `ti.init(arch=ti.cpu)`。
- 为了保证帧率，曲线绘制采用 **Batching** 策略：先在 CPU 端用 `sample_bezier` 算出全部采样点，再一次性传入 `draw_batch_kernel`，由 GPU 并行完成像素写入。
- Spline 模式需要至少 2 个控制点才能生成曲线；控制点越多，分段越细腻。
