# CG-Lab1 Pro-3D Cube Wireframe

## 项目简介

本实验在基础实验（2D 三角形 MVP 变换）的基础上，将几何图元升级为三维正方体，实现了具有透视效果的 3D 立方体线框旋转。

核心内容包括：
- **构建三维几何体**：定义中心在原点、边长为 2 的正方体（8 个顶点，12 条边）
- **模型变换（Model）**：绕 Y 轴和绕 X 轴的旋转矩阵，支持组合旋转
- **视图变换（View）**：基于相机位置的视图矩阵
- **投影变换（Projection）**：透视投影矩阵
- **线框渲染**：遍历 12 条边进行线框绘制，不同面的边使用不同颜色区分深度

## 项目架构

```
Work1_pro/
├── _init_.py        # 模块初始化文件
├── config.py        # 配置参数（窗口、相机、正方体顶点与边、颜色）
├── transform.py     # MVP 矩阵变换核心实现（含 Y 轴和 X 轴旋转）
├── main.py          # 主程序入口与渲染循环
├── cg_work1pro_demo.gif  # 效果展示
└── README.md        # 说明文档
```

| 文件 | 职责 |
|------|------|
| `config.py` | 定义窗口分辨率、相机参数（位置、FOV、近远平面）、正方体 8 个顶点坐标、12 条边索引和颜色常量 |
| `transform.py` | 实现四个变换矩阵：`get_model_matrix_y`（绕 Y 轴旋转）、`get_model_matrix_x`（绕 X 轴旋转）、`get_view_matrix`（视图）、`get_projection_matrix`（透视投影） |
| `main.py` | 初始化 Taichi、创建 GUI 窗口、处理键盘事件、执行 MVP 变换流水线并绘制正方体线框 |

## 实现功能

- **3D 正方体构建**：8 个顶点坐标在 [-1, 1] 范围内，中心位于原点 (0, 0, 0)，边长为 2
- **MVP 矩阵变换流水线**：手动实现 Model、View、Projection 三个变换矩阵并完成矩阵级联
- **双轴旋转**：支持绕 Y 轴和 X 轴的独立旋转，可自由观察立方体的各个面
- **自动旋转**：默认开启绕 Y 轴自动旋转，可通过空格键切换
- **透视投影**：采用透视→正交的分解方法，呈现近大远小的 3D 透视效果
- **NDC 归一化**：齐次坐标除以 w 分量完成透视除法，映射到归一化设备坐标
- **屏幕映射**：将 NDC 坐标 [-1, 1] 映射到屏幕坐标 [0, 1]
- **线框渲染**：用不同颜色绘制正方体前面（绿色）、后面（青色）和连接边（白色），便于区分空间深度

## 运行方式

### 运行程序

在项目根目录（`CG-lab/src`）下执行：

```bash
uv run -m src.Work1_pro.main
```

### 键盘操作

| 按键 | 功能 |
|------|------|
| `A` | 绕 Y 轴逆时针旋转 5° |
| `D` | 绕 Y 轴顺时针旋转 5° |
| `W` | 绕 X 轴向上旋转 5° |
| `S` | 绕 X 轴向下旋转 5° |
| `空格` | 开关自动旋转 |
| `ESC` | 退出程序 |
